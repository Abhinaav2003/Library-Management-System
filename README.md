# 📚 Library Management System

A **Java-based Library Management System** that simplifies the management of books, members, and borrowing activities. This project demonstrates **Object-Oriented Programming (OOP)** concepts, clean code practices, and efficient data handling using Java.

---

## 🚀 Features

- 📖 Add, update, delete, and search books
- 👤 Manage library members
- 🔄 Issue and return books
- 📅 Track issue and return dates
- 📊 Display available and issued books
- 🔍 Search books by title, author, or ID
- ⚡ Simple and user-friendly console interface
- 🛡️ Input validation and error handling

---

## 🛠️ Tech Stack

- **Language:** Java
- **Concepts:** Object-Oriented Programming (OOP)
- **IDE:** IntelliJ IDEA / VS Code
- **JDK:** Java 21+ (Compatible with Java 11+)
